# Task-manager-
Проект под названием "Task manager" включает в себя разработку веб-приложения для определенной цели - управление задачами, так называемый «менеджер задач». 

Это приложение будет удобно в первую очередь для студентов или работающих людей, которым нужно отслеживать количество выполненных задач, помнить об определенных дедлайнах и тп. Все же оно подойдет и для любого пользователя, который хочет записать список покупок или планы на день. 

Какие данные будут храниться. 
Для каждого пользователя будут сохранены его пароль, username, каждому будет присваиваться уникальный id, а также указываться время регистрации. Для каждой задачи будет хранится уникальный id, название и описание, дата создания задачи и ее завершения, а также состояние в котором она находится (завершена или нет).

Можно выделить две таблицы для базы данных: таблица "users" с информацией о каждом пользователе и таблица "tasks" для хранения информации о конкретной задаче. 

Основные действия: возможность создать новую задачу или редактировать старую, создать ей описание, указать: «завершена/не завершена», обозначить срок ее выполнения, удалить задачу, просмотреть список всех задач.
Дополнительные функции: возможность сортировать по дате или завершению; возможность поиска конкретной задачи
