# Индивидуальный проект: Task-manager
Проект под названием **"Task manager"** включает в себя разработку веб-приложения для определенной цели - управление задачами, так называемый «менеджер задач». 

Это приложение удобно в первую очередь для студентов или работающих, которым нужно отслеживать количество выполненных задач, помнить об определенных дедлайнах и тп. Оно подойдхолит и для любого пользователя, который хочет записать планы на день. 

## Основная информация

**Какие данные будут храниться**

Для каждого пользователя сохраняются его пароль, username и имя, присваивается уникальный id, а также указывается время регистрации. Для каждой задачи хранится уникальный id, название и описание, дата создания задачи и ее завершения, приоритет по выполнению (высокий, средний, низкий), а также состояние в котором она находится (завершена или нет).

Создаются две таблицы для базы данных: _"users"_ с информацией о каждом пользователе и _"tasks"_ для хранения информации о конкретной задаче.
[Описание таблиц](https://github.com/qwuzyaa/Task-manager/blob/main/database/description_db.md)

**Основные действия**

Пользователю предоставлена возможность создания задачи (отдельная кнопка), где он указывает её название*, описание и время окончания (* - обзательный параметр), после создания можно изменить опредлелённый параметр (отдельная кнопка), рядом с самой задачей находятся кнопки для смены её статуса и приоритета, а также удаления задачи.

Пользователь может изменить свои данные, указав один из трех параметров, учитывая, что username должен быть уникальным, а поменять пароль можно только указав старый. Есть возможность выхода из аккаунта и его удаление.

Дополнительные функции: статистка задач по следующим категориям: количество всех задач, только выполненные, только активные, только просроченные; поиск задачи по имени; фильтрация по дате окончания, по статусу, по приоритету (низкий, средний, высокий) которая расположена отдельно, вывод списка просроченных задач.


## Используемый стек

- **Язык:** Python 3
- **База данных:** SQLite
- **Backend:** FastAPI
- **Шаблонизатор:** Jinja2
- **Тестирование API:** Swagger / Postman
- **Контроль версий:** Git

Проект выполнен в рамках учебного курса.


## Установка и запуск проекта

1. Клонировать репозиторий
   
   `git clone https://github.com/qwuzyaa/Task-manager.git`
   
   `cd <папка-проекта>`
3. Создать виртуальное окружение
   
   `python -m venv venv`
   
   Активация: `venv\Scripts\activate` (для Windows)
5. Установить зависимости
   
   `pip install -r requirements.txt`
6. Создать базу данных
   
   `cd database` - переходим в папку для создания БД
   
   `python db.py` - запускаем код
   
   `cd ..` - возвращаемся обратно
8. Запусить сервер
   
   `python main.py`
10. Открыть в браузере: http://localhost:8000, для тестирования в Swagger: http://localhost:8000/docs
