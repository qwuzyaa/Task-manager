# Индивидуальный проект: Task-manager
Проект под названием **"Task manager"** включает в себя разработку веб-приложения для определенной цели - управление задачами, так называемый «менеджер задач». 

Это приложение удобно в первую очередь для студентов или работающих, которым нужно отслеживать количество выполненных задач, помнить об определенных дедлайнах и тп. Оно подойдхолит и для любого пользователя, который хочет записать планы на день. 

## Основная информация

**Какие данные будут храниться**

Для каждого пользователя сохраняются его пароль, username и имя, присваивается уникальный id, а также указывается время регистрации. Для каждой задачи хранится уникальный id, название и описание, дата создания задачи и ее завершения, приоритет по выполнению (высокий, средний, низкий), а также состояние в котором она находится (завершена или нет).

Создаются две таблицы для базы данных: _"users"_ с информацией о каждом пользователе и _"tasks"_ для хранения информации о конкретной задаче.
[Описание таблиц](https://github.com/qwuzyaa/Task-manager/blob/main/database/description_db.md)

**Основные действия**

Возможность создать новую задачу или редактировать старую, создать ей описание, указать: «завершена/не завершена», обозначить срок ее выполнения, удалить задачу, просмотреть список всех задач.
Дополнительные функции: возможность сортировать по дате или завершению; возможность поиска конкретной задачи
